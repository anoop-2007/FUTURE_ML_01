# FUTURE_ML_01
# Sales & Demand Forecasting

Predicting sales using XGBoost and Linear Regression on the Superstore Sales dataset.

## Models Used
- XGBoost Regressor
- Linear Regression

## Dataset
Superstore Sales Dataset (Kaggle)

## Results
- XGBoost MAE: $12,155 | RMSE: $16,000 | R²: 0.91
- Linear Regression MAE: $12,882 | RMSE: $16,500 | R²: 0.89
- XGBoost outperformed Linear Regression on all metrics
